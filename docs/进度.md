## 技术笔记

### 交叉编译注意事项

- 板子 glibc 2.30 (Buildroot)，主机 glibc 2.34 → ARM 编译需 `-static`
- 内核 4.9.88，交叉编译器头文件版本较新 → V4L2 结构体布局可能不兼容
- 推送命令: `make push CROSS=arm`
- 运行: `adb shell /root/<程序名> --selftest`

### Makefile 已知问题

- ARM release 模式下使用 `-static` 静态链接
- push 目标会先 clean 再编译 (避免 x86/ARM 混用 .o 文件)
